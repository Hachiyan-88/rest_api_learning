# 用語集（REST API/JSON/Postman）

このページは①～順に登場した重要な用語をまとめています。  
「**自分が後で見返したときに、理解しやすいように**」を目的に整理しています。

---

## 🌐API（Application Programming Interface）

ソフトウェア同士が通信するための窓口  
👉アプリ ⇔ サーバーがデータをやり取りする仕組み

---

## 🌐REST

API設計のルール・考え方の一つ  
特徴：

- URLで資源（リソース）を表す
- HTTPメソッドで操作を表す
- ステートレス（状態を持たない）

---

## 🌐REST API

RESTの考え方に基づいたAPI。

例：

```bash
GET /users
```

👉ユーザー一覧取得

---

## 🌐リソース(Resource)

APIで扱うデータの対象。

例：

- users
- posts
- comments

👉URLの名詞部分

---

## 🌐エンドポイント(Endpoint)

APIにアクセスするURL。

例：

```arduino
https://example.com/users
```

👉APIの入り口

---

## 🌐HTTP

WEB通信のルール  
ブラウザやアプリはHTTPを使い通信を行っている。

---

## 🌐HTTPメソッド

通信の目的を表す。

| **メソッド** | **意味** |
| ----- | ----- |
| GET | 取得 |
| POST | 作成 |
| PUT | 更新 |
| DELETE | 削除 |

---

## 🌐ステータスレコード

通信結果を表す番号。

| **コード** | **意味** |
| 200 | 成功 |
| 404 | 見つからない |
| 500 | サーバーエラー |

---

## 🌐リクエスト(Request)

クライアント→サーバーへ送る要求。

例：  
「ユーザー一覧をください」

---

## 🌐レスポンス(Response)

サーバー→クライアントへ返す結果。

例：  
ユーザーデータ（JSON）

---

## 🌐ヘッダー(Header)

通信に関する追加情報。

例：  

```pgsql
Content-Type: application/json
```

👉データ形式の宣言

---

## 🌐JSON (JavaScript Object Notation)

データを表現するフォーマット。

特徴：

- 人が読める
- 軽量
- 多言語対応

---

## 🌐オブジェクト (JSON)

`キー`と`値`のセット。

```json
{
    "name":"Taro"
}
```

---

## 🌐配列（Array）

複数のデータを並べる構造。

```json
[
    {"name":"Taro"},
    {"name":"Hanako"}
]
```

---

## 🌐ネスト（入れ子構造）

JSONの中にJSONが入る構造のこと。

```json
{
    "user": {
        "name":"Taro"
    }
}
```

---

## 🌐キー（key）

データの名前（ラベル的な）。

```arduino
"name"
```

---

## 🌐値（Value）

キーに対応する中身。

```arduino
"Taro"
```

---

## 🌐Content-Type

送信データの形式を示すヘッダー。

```bash
application/json
```

👉「これはJSONです。」と宣言

---

## 🌐クエリパラメータ

URLの末尾に付ける検索条件。

```bash
/users?id=5
```

👉条件に一致するデータ取得。

---

## 🌐パスパラメータ

URLの一部として指定する値。

```bash
/users/5
```

👉ID 5のユーザー。

---

## 🌐JSONレスポンス

APIから返ってくるJSON形式のデータ。

---

## 🌐Postman

API通信をテストできるツール。

できること：

- リクエスト送信
- レスポンス確認
- ヘッダー確認
- POST送信テスト

---

## 🌐ワークスペース（Postman）

リクエストや設定をまとめて管理する場所。

---

## 🌐コレクション（Postman）

複数のリクエストをまとめたフォルダのようなもの。

---

## 🌐Body

リクエスト送信時のデータ本体。  
POST送信で使用する。

---

## 🌐JSONPlaceholder

無料で使えるテスト用REST API。  
学習・練習用として広く利用されている。
