# REST API & JSON 学習ノート

このリポジトリ（またはフォルダ）は、
**REST API と JSON について学習した内容を、自分用に噛み砕いてまとめる** ためのノートです。

自分が、

- 「その時は分かったつもり」
- 「でも数日後に読むと分からない」

という状態にならないように、
自分なりにかみ砕いて**やさしい言葉・例え話多め**で記録していきます。

---

## 🎯 このノートの目的

- REST APIの考え方を理解できるようになる
- HTTPやJSONとの関係を整理する
- 実装したときに「なぜそう書くのか」を理解できるようになる
- 将来の自分が読み返しても理解できる資料を作る
- 初学者向けに可能な限りかみ砕いてまとめてみる

※ 完璧な定義より「腹落ち」を優先する

---

## 👤 想定読者

- REST APIをこれから学ぶ人向け
- Webアプリの仕組みがまだ曖昧な人向け
- 将来読み返す自分向け

- 教科書っぽく書かない
- 「最初ここが分からなかった」を正直に書く
- 例え話を積極的に使う
- 今は分からないことは「今は分からない」と書く
